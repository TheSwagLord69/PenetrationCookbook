#SQL #MariaDB

Show all databases
```
SHOW DATABASES;
```

Use specific database
```
USE db_name
```
```
USE DATABASE db_name;
```

Show tables in the database
```
SHOW TABLES;
```
```
SHOW FULL TABLES;
```

List all data in table
```
SELECT * FROM administrators;
```
